QUICK START GUIDE
================================================================================

1. LOCAL DEVELOPMENT
   pip install -r requirements.txt
   python run.py
   open http://localhost:8000/docs

2. DOCKER BUILD
   docker build -t talent-api .
   docker run -p 8000:8000 talent-api

3. DOCKER COMPOSE
   docker-compose up -d
   open http://localhost:8000/docs

4. RENDER DEPLOYMENT
   - Push to GitHub
   - Create Render Web Service
   - Set Build Command: pip install -r requirements.txt
   - Set Start Command: gunicorn -w 4 -k uvicorn.workers.UvicornWorker app.main:app
   - Deploy

5. LOAD TESTING
   pip install locust
   locust -f locustfile.py --host http://localhost:8000
